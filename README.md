# Welday Modas

Site estatico da Welday Modas, loja de roupas e confeccao propria em Andradas-MG.

## Publicar no GitHub Pages

1. Crie um repositorio no GitHub.
2. Envie estes arquivos para a raiz do repositorio:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `.nojekyll`
3. No GitHub, abra `Settings` > `Pages`.
4. Em `Build and deployment`, escolha:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Salve e aguarde o GitHub gerar o link do site.
